📚 Programming Assignment 1 - C Language

This directory contains the implementation of Programmming Assignment 1 from the course Algoritmos and Programming Techniques.

The programs were developed using the C programming language as part of the first programming exercises of the course.

The goal of this assgnment is to practice fundamental programming concepts such as input/output operations, variable manipulation, arithmetich calculations, and basic logical reasoning.

🎯 Assignment Objective

The main objective of this assignment is to reinforce core programming concepts in C, including:

- Reading user input
- Printing formatted output
- Working with variables and data types
- Performing Mathematical calculations
- Comparing values
- Organizing simples programs

Each exercise focuses on a specific fundamental concept commonly used in introductory programming courses

📂 Project Sctructure

This Folders contains seven independent programs, each corresponding to a specific exercise from the assignment.

File	            Description
exercicio1.c	Reads a person's name, age, and height and prints the information in a formatted output

exercicio2.c	Reads an employee's salary and calculates the new salary based on a percentage increase

exercicio3.c	Converts age in years to months and days

exercicio4.c	Reads four grades from a student, calculates the average, and determines the highest grade

exercicio5.c	Calculates the average fuel consumption of a car based on distance and fuel used

exercicio6.c	Converts hours into minutes and seconds

exercicio7.c	Calculates a product discount and determines the final price

Each program was written and tested independently.

🛠 Technologies Used

C Programming Language

GCC (GNU Compiler Collection)

Visual Studio Code

▶ How to Compile and Run

To compile a program, use the GCC compiler in the terminal.

Example:

gcc exercicio1.c -o exercicio1
./exercicio1

Repeat the process for any other file in this directory.

Example:

gcc exercicio5.c -o exercicio5
./exercicio5
📖 Concepts Practiced

This assignment focuses on the following C programming fundamentals:

printf() for formatted output

scanf() for user input

Basic data types (int, float, char)

Arithmetic operations

Conditional comparisons

Simple program structure using main()

These concepts represent the first steps in learning structured programming.
